About QNT9 Stock Search

Overview

QNT9 Stock Recommendation System is a microservices-based stock analysis platform built on Azure cloud infrastructure. The system follows Warren Buffett's value investing principles, providing real-time stock analysis, custom watchlists, and automated weekly reports.

Search Capabilities

ISIN (International Securities Identification Number)

12-character alphanumeric code. Example: DE0005140008

WKN (Wertpapierkennnummer)

6-character alphanumeric code used in Germany. Example: 514000

Stock Symbol

Ticker symbol with exchange. Example: DBK.DE

Technology Stack

  • Frontend: FastAPI + HTMX + Tailwind CSS
  • Backend: Python microservices with PostgreSQL
  • Data Sources: Yahoo Finance, Alpha Vantage
  • Infrastructure: Azure AKS, Terraform, HashiCorp Vault
