Stock Recommendation System

Overview

QNT9 Stock Recommendation System is a microservices-based stock analysis platform built on Azure cloud infrastructure. The system follows Warren Buffett's value investing principles, providing real-time stock analysis, custom watchlists, and automated weekly reports.
